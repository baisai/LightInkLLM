# LightInkLLM
LightInkLog and LightInkLua and LightInkMsgPack


# build


## win32

* install vs2012
* open file ./windows/LightInkLLM.sln
* build
* the file output in ./bin/Debug(Release)/

## linux

* install cmake
* mkdir ./Linux/build
* cd ./Linux/build
* cmake ../
* make
* the file output in ./bin/Debug(Release)/
