# LightInkLLMT
LightInkLog and LightInkLua and LightInkMsgPack and LightInkTinySocket


# build


## win32

* install cmake and vs2012
* mkdir ./build
* cd ./build
* cmake ../project
* the vs project in ./build

## linux

* install cmake
* mkdir ./build
* cd ./build
* cmake ../project
* make
* the file output in ./bin/Debug(Release)/
